以下是一个简单的 HTML 代码示例，实现了一个导航页，用户可以输入内容，并选择多个搜索引擎进行搜索，点击搜索按钮后，会在新的标签页打开选定搜索引擎的搜索结果页面：

```
htmlCopy code<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>搜索引擎导航</title>
</head>
<body>
    <h1>搜索引擎导航</h1>
    <form id="searchForm">
        <label for="searchQuery">输入内容:</label>
        <input type="text" id="searchQuery" name="searchQuery">
        <br>
        <label for="searchEngines">选择搜索引擎:</label>
        <select id="searchEngines" name="searchEngines" multiple>
            <option value="https://www.google.com">Google</option>
            <option value="https://www.bing.com">Bing</option>
            <option value="https://www.yahoo.com">Yahoo</option>
        </select>
        <br>
        <button type="submit">搜索</button>
    </form>

    <script>
        document.getElementById("searchForm").addEventListener("submit", function(event) {
            event.preventDefault(); // 阻止表单提交

            var searchQuery = document.getElementById("searchQuery").value;
            var selectedEngines = Array.from(document.getElementById("searchEngines").selectedOptions).map(option => option.value);
            
            // 在新标签页中打开选定搜索引擎的搜索结果页面
            selectedEngines.forEach(function(engine) {
                window.open(engine + "/search?q=" + encodeURIComponent(searchQuery), "_blank");
            });
        });
    </script>
</body>
</html>
```

这个 HTML 页面包含一个表单，用户可以输入内容并选择多个搜索引擎。当用户点击搜索按钮时，JavaScript 代码会捕获表单提交事件，并根据用户选择在新的标签页中打开相应搜索引擎的搜索结果页面。
